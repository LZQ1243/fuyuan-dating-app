## 🚀 快速开始

### 开发环境

```bash
# 1. 安装依赖
cd backend && npm install
cd ../frontend-react && npm install
cd ../fuyuan-taro && npm install

# 2. 启动服务
npm run dev  # 各个目录下执行
```

### 生产部署

```bash
# 使用自动化脚本
./deploy-universal.sh  # Linux/macOS
.\deploy-universal.ps1 # Windows

# 或使用 Docker
docker-compose -f docker-compose.universal.yml up -d
```
