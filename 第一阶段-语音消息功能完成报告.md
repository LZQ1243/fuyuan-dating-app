## 📡 API文档

### 上传语音
```
POST /api/voice/upload
Content-Type: multipart/form-data

Body:
- audio: (file) 音频文件
- duration: (number) 时长(秒)

Response:
{
  "success": true,
  "data": {
    "id": "uuid",
    "filePath": "/uploads/audio/uuid.mp3",
    "size": 1024000,
    "mimeType": "audio/mpeg",
    "duration": 30,
    "uploadedAt": "2026-03-19T00:00:00.000Z"
  }
}
```

### 删除语音
```
DELETE /api/voice/:id
Authorization: Bearer <token>

Response:
{
  "success": true,
  "message": "删除成功"
}
```

## 🎯 使用示例

### 在聊天页面中使用
```tsx
import VoiceRecorder from '@/components/VoiceRecorder'
import VoicePlayer from '@/components/VoicePlayer'

const ChatDetail = () => {
  const [messages, setMessages] = useState([])

  const handleVoiceSend = async (audioData) => {
    const result = await uploadVoice(audioData)
    setMessages(prev => [...prev, {
      type: 'voice',
      ...audioData,
      url: result.data.filePath,
      fromSelf: true
    }])
  }

  return (
    <View>
      {messages.map(msg => (
        msg.type === 'voice' ? (
          <VoicePlayer message={msg} />
        ) : (
          <Text>{msg.content}</Text>
        )
      ))}
      
      <VoiceRecorder onSend={handleVoiceSend} />
    </View>
  )
}
```
