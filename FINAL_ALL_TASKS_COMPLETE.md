## 📋 配置中心详细说明

### 公开配置接口
```
GET /api/config/public
```
前端自动调用，无需认证，返回：
- API基础URL
- 功能开关
- 文件上传限制

### 私有配置接口
```
GET /api/config          # 管理员获取所有配置
PUT /api/config          # 管理员更新配置
PUT /api/config/batch    # 管理员批量更新配置
POST /api/config/reset    # 管理员重置配置
```

### 前端配置获取流程
```
1. 前端启动
2. 调用 GET /api/config/public
3. 获取API基础URL
4. 更新axios实例的baseURL
5. 后续请求使用新URL
```

### 配置中心管理流程
```
1. 管理员登录
2. 导航到"系统配置"
3. 选择配置分类标签页
4. 编辑配置项
5. 点击保存
6. 前端自动获取新配置
```
