## 🚀 重新启动开发流程

### 1. 环境准备

#### 1.1 清理工作区
```bash
# 清理npm缓存
cd backend
rm -rf node_modules
npm cache clean --force

# 清理前端缓存
cd frontend-react
rm -rf node_modules
npm cache clean --force

# 清理小程序缓存
cd fuyuan-taro
rm -rf node_modules
npm cache clean --force
```

#### 1.2 重新安装依赖
```bash
# 后端依赖
cd backend
npm install --legacy-peer-deps

# 前端依赖
cd ../frontend-react
npm install --legacy-peer-deps

# 小程序依赖
cd ../fuyuan-taro
npm install --legacy-peer-deps
```

### 2. 启动所有服务

#### 2.1 启动数据库
```bash
# MongoDB
docker run -d -p 27017:27017 --name mongodb

# Redis
docker run -d -p 6379:6379 --name redis

# RabbitMQ
docker run -d -p 5672:5672 --name rabbitmq
```

#### 2.2 启动应用
```bash
# 后端服务 (新终端)
cd backend
npm run dev

# 前端服务 (新终端)
cd ../frontend-react
npm run dev

# 小程序开发工具 (新终端)
cd ../fuyuan-taro
npm run dev:weapp

# 总管理后台 (新终端)
cd ../admin
npm run dev

# 红娘管理后台 (新终端)
cd ../matchmaker-admin
npm run dev
```
