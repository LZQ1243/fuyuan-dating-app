## 🎯 核心技术实现

### 1. 智能匹配算法

```javascript
// 多维度加权计算
- 残疾等级: 40%权重 (同等级+10分, 相邻+5分, 差距大-5分)
- 残疾类型: 35%权重 (同类型+8分, 互补型+12分, 排斥型-20分)
- 地理位置: 15%权重 (同城+5分, 同区+10分)
- 年龄范围: 10%权重 (范围内+5分, 超出-2分)

// 最终得分 = 各维度得分 × 权重
// 排序返回Top N用户
```

```
WebSocket架构:
- Socket.IO服务器
- 在线用户状态管理
- 消息实时推送
- 离线消息存储
- 已读状态同步
```

### 3. 敏感词过滤


## 🚀 快速启动

### 开发环境

```bash
# 1. 启动数据库服务
cd docker
docker-compose up -d mongodb redis rabbitmq

# 2. 启动后端
cd ../backend
npm install
npm run dev

# 3. 启动前端
cd ../frontend
npm install
npm run dev:h5

# 4. 启动管理后台
cd ../admin
npm install
npm run dev
```

### 生产部署

```bash
# 一键部署
cd docker
docker-compose up -d
```
