## 技术实现

### 1. 粉色渐变背景
```scss
.bg-decoration {
  background: linear-gradient(180deg, #FF8E8C 0%, #FF6B6B 100%);
  border-radius: 0 0 200px 200px;
}
```

### 2. 卡片式表单
```scss
.form-section {
  background-color: #ffffff;
  border-radius: 24px;
  padding: 60px 30px;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.1);
}
```

### 3. 装饰性几何元素
```scss
.deco-circle {
  width: 12px;
  height: 12px;
  border-radius: 50%;
  background-color: rgba(255, 255, 255, 0.5);
}

.deco-line {
  width: 80px;
  height: 2px;
  background-color: rgba(255, 255, 255, 0.3);
}
```

### 4. 响应式设计
```typescript
<ScrollView scrollY className='form-container'>
  {/* 表单内容 */}
</ScrollView>
```
