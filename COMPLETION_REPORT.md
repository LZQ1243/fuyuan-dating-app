### 方式一: 一键启动 (推荐)

```cmd

```

#### Linux/Mac:
```bash

```

## 📋 技术栈

### 前端
- **框架**: uni-app + Vue3
- **UI库**: uView Plus
- **状态管理**: Pinia
- **网络请求**: uni.request

### 后端
- **语言**: Node.js
- **框架**: Express
- **数据库**: MongoDB + Redis
- **消息队列**: RabbitMQ
- **实时通信**: Socket.IO

### 管理后台
- **框架**: Vue3 + Element Plus
- **构建工具**: Vite
- **图表库**: ECharts

### 部署
- **容器化**: Docker + Docker Compose
- **Web服务器**: Nginx
