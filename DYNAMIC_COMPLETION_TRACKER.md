## 十、部署指南

### 快速启动

```bash
# 启动后端
cd backend && npm start

# 启动前端
cd frontend-react && npm run dev

# 启动小程序
cd fuyuan-taro && npm run dev:weapp

# 启动管理后台
cd admin && npm run dev

# 启动红娘后台
cd matchmaker-admin && npm run dev
```

### Docker 部署

```bash
# 启动所有服务
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f backend

# 停止服务
docker-compose down
```
